智能记账软件使用手册

首次使用，请仔细阅读以下内容

⚠️ 重要提醒

  • 数据只保存在你的设备上，不会上传到任何服务器
  • 清除浏览器数据会导致所有记录丢失！
  • 换手机/电脑无法自动同步数据
  • 忘记密码可用密保问题找回
  • 忘记密保答案将无法恢复数据！

📋 快速开始

  1. 记支出：点击「支出」→ 输入金额 → 选择分类 → 保存
  2. 记收入：点击「收入」→ 输入金额 → 选择分类 → 保存
  3. 查看统计：切换「收支趋势」「分类统计」「日历视图」
  4. 设置预算：在预算区域输入金额，查看使用进度

💾 数据备份（必看！）

强烈建议每次记账后都备份！

JSON 格式备份（推荐，数据完整）：

  1. 选择「JSON 格式」
  2. 点击「导出 JSON」按钮
  3. 发送到微信文件传输助手
  4. 或保存到 iCloud/百度网盘

CSV 格式备份（方便查看）：

  1. 选择「CSV 格式」
  2. 点击「导出 CSV」按钮
  3. 可用 Excel、WPS 打开查看账目

⚠️ 如果不备份，清除浏览器数据后记录将全部丢失！

🔐 密码与密保

  • 忘记密码：点击「忘记密码」→ 回答密保问题 → 重置密码
  • 密保答案区分大小写，请牢记
  • 建议把密保答案写在安全的地方

💡 隐私说明

每个人使用都是独立的，你的数据只存在你自己的设备上。 即使别人使用同一个软件，也完全看不到你的记录。
